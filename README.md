# 📍 SkyLink AI Logistics - Smart Map Navigator

Prolog Logic Engine နှင့် Google Maps API ကို အသုံးပြု၍ မြန်မာနိုင်ငံရှိ မြို့များအကြား အတိုဆုံးနှင့် အကောင်းဆုံးလမ်းကြောင်းကို ရှာဖွေပေးသည့် Full-Stack Web Application ဖြစ်ပါသည်။

---

## 🛠️ Tech Stack & Architecture

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
* **Backend Engine:** Python (FastAPI / Flask) & Prolog (Logic-based Shortest Path Solver)
* **Database/Data:** Shortest path node relationships mapped logically.

---

## 📂 Core Files

* `Smartmap.html` - Minimalist user dashboard & interactive map layout.
* `script.js` - Dynamic UI logic, route fetching, and API integration.
* `server.py` - Backend server connecting JavaScript requests to the Prolog logic core.
* `pathfinder.cpp` - Core routing algorithm engine.
